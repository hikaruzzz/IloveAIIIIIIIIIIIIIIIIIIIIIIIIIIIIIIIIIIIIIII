# A-Stock-Data Skill

当用户调用 `/share-a-stock-data` 或需要获取 A股 / 港股 / 美股 / 贵金属 / 大宗商品 行情数据时，按以下规则执行。

## 支持的市场

- A股
- 港股
- 美股
- 贵金属（伦敦金/银、COMEX 黄金/白银等）
- 大宗商品（原油、铜、铝等）

## 支持的数据类型

实时行情、历史K线、财务数据、公司公告、新闻资讯、研报、资金流向、龙虎榜、板块数据、指数数据、宏观经济、期货数据

## 操作流程

### 获取行情数据

用户请求获取某标的行情数据时：

1. 确认标的代码和市场
2. 根据用户需要的数据类型，从底层接口获取对应数据
3. 以结构化格式返回：当前价格、技术指标（均线/MACD/RSI 等）、基本面数据
4. 如用户请求期货相关数据（持仓量、多空比、基差等），一并获取

### 数据返回格式

返回数据时使用统一格式：

```
标的：{名称}（{代码}）
市场：{市场}
获取时间：{当前时间}

【价格数据】
当前价格：xxx
涨跌幅：xxx

【技术指标】
MA5：xxx
MA10：xxx
MA20：xxx
MACD：DIF xxx / DEA xxx / 柱 xxx（金叉/死叉）
RSI(14)：xxx

【期货数据】（如适用）
前20席位多空持仓比：xxx（近5日趋势：xxx）
持仓量OI变化率：xxx
期货-现货基差：xxx
期货成交量/5日均量：xxx

【消息面】
（相关新闻和事件摘要）
```

## 关键约束

1. 返回数据时标注数据获取时间
2. 如果某项数据无法获取，明确标注「暂无数据」而非留空
3. 技术指标计算时说明周期（日线/小时线等）
